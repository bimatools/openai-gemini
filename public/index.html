This is demo instance of Gemini ➜ OpenAI API transforming proxy!
Running serverless on Vercel.

You can try it with: 

curl ${origin}/v1/chat/completions \
  -H "Authorization: Bearer $YOUR_GEMINI_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
      "model": "gpt-3.5-turbo",
      "messages": [{"role": "user", "content": "Hello"}],
      "temperature": 0.7
      }'
